# 旧版MC音效包制作工具

简单的小工具，能够使用GUI快速生成JE/PE版的音效包（本质上就是一个基本上只存放了音频文件的资源包）。

已经停更...多久来着？

## 旧版停更说明

本来是打算写完一系列自定义控件（美化界面用）后，换用新控件再更新的。但是嘛...软件原本使用.NET Framework开发，虽然新控件开发也取得了我自认为非常好的成果，比如给类型的字段/属性(`Property`)的定义处添加一个自定义特性(`Attribute`)，就可以自动生成修改该字段/属性的控件集合。然而，想到还有`TextBox`、`ComboBox`、`ListBox`等等一大堆控件等着我去重写，emm...

总而言之，后来我就打算换用.NET Core重构。@但是拖了很久都只有新建文件夹@

然后，应该还会换用json保存项目数据。
