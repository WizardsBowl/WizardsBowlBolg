# 巫师之碗简介

巫师之碗(**Bowl of Wizard**)是我(**WizardsBowl**/**碗里巫云**)的个人博客。

## 主站和副站

本站(巫师之碗)可以认为是本人博客的主站，此外还有一个副站([巫碗子](https://www.wzb233.com))，至于二者之间的区别嘛...其实还真差不多？@@反正都是我的个人博客@@

## 主站是干啥的

网站使用静态站点生成器[VitePress](https://vitepress.dev/zh/)生成，源代码托管在[GitHub](https://github.com/WizardsBowl/WizardsBowlBolg)上，静态资源托管在[CloudFlare](https://developers.cloudflare.com)上。

我在这里大概会发布一些我自己制作的软件工具，或是记录一些解决问题的经历，供大家参考。

## 关于主站的名称

由于一个贫血错误，网站的GitHub仓库名称变成了`WizardsBowlBolg`，所以你也可以叫它**BoBo**或**啵啵**。
